VITAの転送ファイル制限にあっているかどうかチェックする鶴です
VITAのセーブフォルダにおいて実行、
LISTでファイルリスト取得、右のタブでステータス表示
FIXでNEEDのやつだけ、それっぽいファイル名になるよう修正されます

//ファイル名ステータス
OK 問題なし
NEED UPPER STRING 大文字化が必要
NEED FIX(VITA8.3) VITAの8.3制限になるよう修正が必要
UNIX DOTFILE(IGNORE) ファイル先頭文字が.のもの,UNIXの隠しファイル,FIX修正対象外(VITAの方でも認識しない)

//FIXボタン
NEED UPPER STRING 大文字化される
NEED FIX(VITA8.3)  VITAの8.3制限になるよう修正
N1の箇所がかぶる場合N1の番号をふやしてリネームします(N1〜N9、更にかぶる場合1文字削ってN00〜N99まで)
[半角英数,ハイフン,アンダーバー6文字以内]N1
[半角英数,ハイフン,アンダーバー6文字以内]N1.[拡張子3文字以内,半角英数,ハイフン,アンダーバー]

仕様、ファイル名の正規表現
Regex VITA＿FILE_NAME_MASK= new Regex"^[A-Z0-9\-_]{1,8}(\.[A-Z0-9\-_]{1,3})?$"
